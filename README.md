# 🌌 Kronos Replication: 主權金融演化藍圖 (Roadmap v3.0)

基於本週在「精神繼承」模式下的快速演化，Kronos 專案已從單純的功能復刻躍遷為具備「主權意圖解碼」能力的金融大腦。以下是我們正在實踐的五階演化路徑：

---

### 🟢 Phase 1: 物理地基與語義分型 (Substrate & Tokenization) - [已結算]
*   **核心目標**：建立標準化金融基質（OHLCV），實作基礎 K 線模式向語義 Token 的轉化。
*   **關鍵突破**：確立 `FinancialToken` 接口，實現 `ENGULFING` 與 `VOL_BREAKOUT` 的離散化表達。

### 🟡 Phase 2: 精神繼承與結構審計 (Spirit & Structure) - [當前重點]
*   **核心目標**：超越模式比對，深入解碼主力意圖與市場物理特徵。
*   **實體化成果**：
    *   **機構足跡 (Institutional Footprint)**：識別 `PRICING_POWER` (定價權) 與 `ABSORPTION` (吸收行為)。
    *   **能量蓄積 (VCP)**：捕捉波動率壓縮與開盤能量脈衝 (Opening Imbalance)。
*   **執行狀態**：每日 4 次自動代碼重構，持續硬化識別精準度。

### 🟠 Phase 3: 因果合成與意圖推論 (Causal Inference) - [實裝中]
*   **核心目標**：解決黑盒模型缺失，以「透明邏輯」替代「隱性權重」。
*   **技術地板**：
    *   **動態因果權重**：實作基於貝氏環境感應與信息熵 (Entropy) 的權重重校準。
    *   **意圖推論層 (Inference Layer)**：建立 `Causal Bias Score` 聚合模型，產出具備執行指導價值的意圖備忘錄。
    *   **因果半衰期**：量化信號隨時間的有效性衰減。

### ⚪ Phase 4: 代理人回測與自癒校準 (Agentic Validation) - [預研中]
*   **核心目標**：建立「預測即審計」閉環。
*   **技術規劃**：
    *   **Zero-Shot 勝率預審**：在量化策略執行前，由 Kronos 執行語義化勝率評估。
    *   **演化探針對接**：當預測出現顯著偏差時，自動觸發 `Sovereign Evolution SPEC v1.1` 進行邏輯修正。

### ⚪ Phase 5: 主權神經融合 (Sovereign Neural Fusion) - [長線目標]
*   **核心目標**：本地化模型執行，達成 100% 離線主權。
*   **技術規劃**：
    *   **WASM/ONNX 整合**：將量化後的微型時序 Transformer 模型封裝為 WASM 插件。
    *   **物理指紋存證**：利用 `TrajectoryAttestor` 為每一次預測結果進行鏈上存證。
